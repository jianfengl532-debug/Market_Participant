🐧 Alpine Linux 命令速查手册

轻量 · 安全 · 高效的日常运维命令参考 —— 适用于 OpenRC + apk 环境

⚡ 开关机与基础操作

操作命令说明
重启系统reboot立即重启
关机 (halt)halt停止系统，不切断电源（部分环境适用）
关机 (poweroff)poweroff推荐，安全关机并切断电源
清屏clear 或 Ctrl+L清除终端屏幕内容

📦 包管理 — apk Alpine Package Keeper

操作命令示例备注
更新仓库索引apk update安装前建议执行
搜索软件包apk search <关键词>如 apk search python
安装软件包apk add <包名>可多个: apk add vim curl
安装（不缓存）apk add --no-cache <包名>适合容器，减小体积
卸载软件包apk del <包名>删除已安装包
升级所有包apk upgrade全局升级已安装软件
升级单个包apk upgrade <包名>只升级特定包
列出已安装包apk info查看所有已装软件
查看包详情apk info -a <包名>版本、依赖、描述
查询文件归属apk info --who-owns <文件路径>例: apk info --who-owns /sbin/reboot
下载apk文件apk fetch <包名>仅下载不安装

🔧 服务管理 — OpenRC

操作命令说明
启动服务rc-service <服务名> start例: rc-service nginx start
停止服务rc-service <服务名> stop
重启服务rc-service <服务名> restart
查看状态rc-service <服务名> status
重载配置rc-service <服务名> reload优雅重载 (支持的服务)
列出所有服务rc-service --list显示所有可用服务
添加开机自启rc-update add <服务名> default加入默认运行级别
移除开机自启rc-update del <服务名> default
查看运行状态rc-status显示当前级别服务状态
查看所有运行级别rc-status -l列出可用级别(boot/default/shutdown等)
📌 常用运行级别：default (日常运行)、boot (系统启动核心)、shutdown (关机时动作)。通常服务加入 default 即可。

🌐 网络配置与常用命令

操作/说明命令 / 配置
查看所有网卡信息ip a 或 ip addr show
测试连通性ping <IP/域名> (按 Ctrl+C 停止)
查看路由表ip route show 或 route -n
配置静态IP编辑 /etc/network/interfaces
重启网络服务rc-service networking restart
 
# 静态IP配置示例（/etc/network/interfaces）
auto eth0
iface eth0 inet static
    address 192.168.1.150
    netmask 255.255.255.0
    gateway 192.168.1.1
    dns-nameservers 8.8.8.8

💾 配置持久化 (无盘模式 / lbu) local backup

⚠️ Alpine Linux 默认无盘模式（如安装到USB或运行在RAM）不会自动保存修改。以下命令使配置重启后仍然生效。
操作命令说明
将文件/目录加入备份列表lbu add <文件/目录>例如: lbu add /etc/network/interfaces
提交备份(.apkovl)lbu commit生成 overlay 文件，存储到引导介质
查看当前备份文件列表lbu list显示待备份的内容
查看备份状态/差异lbu status显示哪些文件变更但未加入备份
删除备份中的某个文件lbu delete <路径>不再备份该文件
指定备份存储设备lbu commit <设备或目录>例: lbu commit /media/usb
📌 典型持久化流程：修改配置 → lbu add 修改过的文件或目录 → lbu commit → 重启后配置保留。

📌 快速配置案例

# 1. 安装并自动启用 ssh 服务
apk add openssh
rc-update add sshd default
rc-service sshd start

# 2. 设置 hostname
echo "my-alpine" > /etc/hostname
hostname my-alpine

# 3. 配置 apk 国内镜像加速（可选）
sed -i 's/dl-cdn.alpinelinux.org/mirrors.aliyun.com/g' /etc/apk/repositories
apk update
🧰 说明：Alpine 不同于 systemd 发行版，使用 OpenRC + rc-service 管理服务，轻巧且稳定。大部分命令与其他 Linux 通用（ls, cd, grep, ps 等均可用）。
