十七、复盘速记例句（可直接复制）

  1. UPT + BRK → L SP
    趋势向上+有效突破，轻仓做多
  2. DNT + BKD → Avoid L
    下跌趋势破位，禁止做多
  3. FUD + DUMP → CFK High Risk
    空头恐慌砸盘，抄底属于接飞刀高风险
  4. No GOV INT → DIS Market
    无政府干预，市场走势分化
  5. DCA on PB (Small SZ)
    波段回调分批低吸，控制轻仓
  6. FOMO Rally → STR Tap Out
    情绪拉涨行情，逢高分批止盈
  7. 5-Wave UPT CYCL done → LAD OUT
    完整五浪上涨周期走完，阶梯分批离场
  8. Liquidity Hunt near SUP → Watch Fake BKD
    支撑附近扫止损，警惕假破位
  9. DIP / PB → BTD (SML SZ)
    出现小幅浅回调，执行逢低吸纳，轻仓介入
  10. PD / FUD / DUMP → BTP (High Risk / CFK)
    市场恐慌踩踏或主力砸盘时抄底，属于高风险接飞刀，严控仓位或等待右侧确认
  11. Only if UPT CONF → L SP
    唯有上涨趋势确认，才可轻仓做多
  12. Only after CONF → open pos
    确认后方可开仓
  13. Trade within RNG only
    只在震荡区间内部交易
  14. unless CPR appears → No BTP
    除非出现割肉投降信号，否则不抄底
  15. DNT + BKD → Avoid L / only SML S allowed
    下跌破位，禁止做多，只允许轻仓做空
