  1. If TD or RNG → PANIC; If RLY → HC
    行情下跌/震荡→谨慎恐慌；反弹拉升→持币观望
  2. No GOV INT → DIV
    无政府干预，行情分化各行其是
  3. If DUMP → CFK (High Risk)
    恐慌砸盘抄底=接飞刀，高风险
  4. If DIP → BTD (Small Pos)
    小幅回调轻仓低吸
