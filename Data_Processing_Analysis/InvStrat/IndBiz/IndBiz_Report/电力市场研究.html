中国电力市场投资环境与重点企业分析报告

🏭 产业分析 📅 2026年7月20日

一、宏观投资环境总览

1.1 电力市场改革进入深水区

2026年是中国电力市场化改革的里程碑之年。2026年2月，国务院办公厅印发《关于完善全国统一电力市场体系的实施意见》（国办发〔2026〕4号），明确了"十五五"期间全国统一电力市场体系建设的总体方向与路径。这是继2015年"中发9号文"之后推动电力市场化改革的又一重要纲领性文件。

改革的核心目标清晰：到2030年，基本建成全国统一电力市场体系，各类型电源和除保障性用户外的电力用户全部直接参与电力市场，市场化交易电量占全社会用电量的70%左右；到2035年，全国统一电力市场体系全面建成。

当前，电力市场"1+6"基础规则体系已构建完成。省级电力现货市场已基本实现全覆盖，现货市场发现价格、调节供需的作用持续加强。分时电价正从行政划分向市场形成过渡，多地已取消固定分时电价机制。输配电价也进入第四监管周期，告别单一用电量计费模式，升级为电量、容量、可靠性与跨区互济的复合计价体系。

对投资者的启示：电力企业的盈利模式正从"计划定价"全面转向"市场定价"，电价波动将成为常态，企业的市场竞争力和成本控制能力将决定其盈利水平。

1.2 供需格局：总体平衡，局部偏紧

从需求端看，2026年上半年全社会用电量累计50999亿千瓦时，同比增长5.3%。中电联预计2026年全年全社会用电量同比增长5%–6%。值得高度关注的是，AI和算力正成为电力需求的重要增量——2025年全国数据中心用电量约1700亿千瓦时，近3年平均增长率约39.5%。预计到2030年，数据中心用电量占比将从2025年的1.89%提升至超过5%。

从供给端看，截至2026年5月底，全国发电装机容量达到40.1亿千瓦。预计到2026年底，全国发电装机容量将达到43亿千瓦左右，其中非化石能源发电装机27亿千瓦，占总装机比重约63%，煤电装机比重进一步降至31%左右。

2026年全国电力供需呈总体平衡态势，但夏季局部地区平衡偏紧。在厄尔尼诺气候影响下，制冷负荷提升叠加燃料价格短期易涨难跌，江苏、广东等用电大省电价已出现显著反弹。

对投资者的启示：电力供需偏紧格局为电价提供支撑，火电的"兜底"价值在市场化定价体系下有望得到更充分体现。

1.3 政策红利：容量电价与电网投资

两大政策变量值得重点关注：

其一，煤电容量电价全面上调。 煤电容量电价回收固定成本比例由30%左右提升至不低于50%，叠加辅助服务市场扩容，火电收入确定性明显增强。这从根本上改变了火电企业的盈利模式——从单一的"发电赚钱"转向"容量保底+电量弹性"的双轮驱动。

其二，"十五五"电网投资规模空前。 "十五五"期间全国电网固定资产投资预计达到5万亿元以上，较"十四五"同期增长超80%。重点聚焦特高压、智能电网和新能源并网等领域。电网投资的大幅扩张将为电力设备、储能、智能电网等产业链带来系统性机会。

1.4 "算电协同"开辟新赛道

2026年，"算电协同"首次写入政府工作报告。国家严格限制算力园区绿电占比，全国多个算力枢纽节点加速推进算电协同项目落地。据机构预测，算电协同对应总投资规模约为2万亿元，"十五五"期间年复合增长率约42%。

这一趋势意味着：算力正在主动寻找最优质、最清洁、最经济的电力资源。拥有区位优势（靠近算力枢纽）、绿电资源丰富或具备综合能源服务能力的企业，将迎来价值重估。

二、国电电力（600795）：国家队龙头的转型之路

2.1 企业定位与规模

国电电力是国家能源集团旗下的核心电力上市公司，定位于集团常规能源整合平台。经过"十四五"发展，公司控股装机已超过1.2亿千瓦，管理装机4426万千瓦，煤炭产能超2600万吨/年，产业分布在23个省（区、市）以及印尼等国家和地区。

截至2026年3月31日，公司控股装机容量12794.91万千瓦，其中火电8267.3万千瓦、水电1519.06万千瓦、风电1062.23万千瓦、光伏1946.32万千瓦。

2.2 财务表现：短期承压，长期韧性

2025年全年： 实现营业收入1702.44亿元，同比下降4.99%；归母净利润71.61亿元，同比下降27.15%；但扣非归母净利润67.76亿元，同比大幅增长45.23%。营收下降主要受售电单价下降影响，但扣非利润大幅增长反映出主业经营质量的实质改善。

2026年一季度： 实现营业收入391.69亿元，同比下降1.62%；归母净利润13.98亿元，同比下降22.79%。一季度公司完成上网电量1004.26亿千瓦时，同比增长5.78%，其中火电同比增长6.51%、光伏同比增长16.14%。但平均上网电价395.48元/千千瓦时，同比下降约7.04%。

核心矛盾：电量增长但电价承压。 公司在产能扩张带动下电量实现稳健增长，但受各省年度长协电价明显调整影响，电价回落拖累了营收和利润表现。

2.3 发展战略："三线推动、三维发力"

公司明确提出"十五五"战略目标：

  • 新增新能源装机1850万千瓦、火电装机640万千瓦
  • 所有火电机组具备20%深度调峰能力
  • 争取在2029年实现碳达峰

战略路径可概括为"三线推动"：开发优质新能源、发展新一代高效煤电、抢先布局战新产业。2026年公司计划固定资产投资517亿元，其中新能源189亿元、水电109亿元、火电165亿元。

从电源结构看，公司正处于水火绿多点装机增长的阶段。大渡河流域水电站进入密集投产期，2025年计划投产136.5万千瓦、2026年计划投产215.5万千瓦。新能源方面，2026年计划投产新能源装机677万千瓦。

2.4 投资价值研判

优势：

  • 煤电一体化成本优势显著。 背靠国家能源集团，自有煤炭产能超2600万吨/年，有效对冲煤价波动。
  • 多能互补的电源结构。 火电、水电、风电、光伏全面布局，在电力市场化定价中具备更强的抗风险能力。
  • 高分红承诺锚定稳健价值。 2025年拟每股分红0.241元，分红金额合计43亿元。公司在行业中的龙头地位和持续高比例分红具备长期投资价值。
  • 容量电价政策直接受益。 火电装机规模全国领先，容量电价上调将带来显著的固定收益增厚。

风险：

  • 电价下行压力短期内难以根本扭转，2026年全市场电价仍处于筑底阶段。
  • 新能源装机增速加快但消纳压力和电价下行风险并存，风电利用率已出现下滑。
  • 2026年一季度经营活动现金流净额同比下降36.88%，主要系售电均价下降及电网结算周期影响。

三、宁波能源（600982）：地方国企的精细化转型样本

3.1 企业定位与业务结构

宁波能源是宁波市属国有能源企业，主要从事热电联产、生物质发电、抽水蓄能和综合能源服务，以及能源类相关金融投资等业务。与国电电力的"大而全"不同，宁波能源走的是"专而精"的区域性能源服务商路线。

3.2 财务表现：营收收缩，盈利提质

2025年全年： 实现营业收入37.87亿元，同比下降10.72%；但归母净利润3.48亿元，同比增长22.09%；扣非归母净利润3.17亿元，同比大幅增长54.57%。

公司呈现出"营收收缩、盈利提质"的鲜明特征——主动收缩贸易相关业务，进一步聚焦高毛利的实体能源产业。

2026年一季度： 实现营业收入10.70亿元，同比增长9.78%；归母净利润7251.81万元，同比大幅增长71.60%。扣非净利润8606.12万元，同比增长22.23%。经营活动现金流净额1.64亿元，同比增长133%。

盈利大幅增长主要得益于毛利增长及财务费用下降（财务费用同比降22.11%）。

3.3 发展战略：向低碳智慧综合能源服务商转型

公司"十五五"战略清晰聚焦三大核心赛道：

  1. 清洁能源项目投资运维
  2. 零碳场景综合服务
  3. 生物质能高值化利用

公司正从传统热电联产向"国内一流区域型低碳（零碳）智慧综合能源服务商"转型升级。

在具体业务布局上，公司通过"开发+收购"双轮驱动模式加快风光储充项目落地。同时密切跟踪国家算电协同战略导向，以能源供应商、基础设施配套方的定位切入，稳健探索合规可持续的商业模式。宁波市作为国家算电协同试点城市，为公司提供了独特的区位政策红利。

此外，公司积极布局新质生产力领域，参股投资油页岩制油项目，研究生物质气化耦合绿电制氢制绿甲醇等前沿技术。

3.4 投资价值研判

优势：

  • 盈利质量持续改善。 主动收缩低毛利贸易业务，聚焦高毛利的能源实体产业，扣非净利润增速远超营收增速。
  • 区域垄断优势。 作为宁波市属国有能源平台，在区域热电联产、综合能源服务领域具备天然的资源壁垒。
  • 算电协同区位红利。 宁波作为国家算电协同试点城市，公司有望在数据中心供能、绿电直供等新赛道中率先受益。
  • 经营效率提升显著。 期间费用管控效果明显，管理费用和销售费用均同比下降，财务费用大幅下降。

风险：

  • 体量较小，抗风险能力相对有限。 与国电电力千亿级营收相比，宁波能源规模偏小，面对燃料价格大幅波动时缓冲空间有限。
  • 业务集中于区域市场。 业务范围主要集中在宁波及周边地区，区域经济波动对业绩影响较大。
  • 转型效果尚需验证。 向综合能源服务商转型的方向正确，但新业务的规模化盈利仍需时间。

从机构视角看，多家券商在2026年中期策略中均推荐关注国电电力。核心逻辑在于：火电盈利筑底、板块估值修复可期，煤电一体化企业凭借自有煤源降低煤价波动影响，有望更充分分享电价上涨收益。

宁波能源虽然未被主流券商列为行业重点推荐标的，但其"收缩低毛利、聚焦高毛利"的战略转型成效显著，盈利质量持续改善，在区域性综合能源服务赛道中具备差异化竞争能力。

五、风险提示

  1. 电价下行风险。 电力市场化改革深化过程中，电价仍面临下行压力，2026年各省年度长协电价已接近下限。
  2. 燃料成本波动风险。 2026年环渤海动力煤价格呈持续上涨趋势，二、三季度火电企业成本端预计承压。
  3. 新能源消纳风险。 风电、光伏利用率均呈现持续下滑趋势，新增装机的高速增长可能进一步加剧消纳压力。
  4. 政策推进不及预期风险。 全国统一电力市场体系建设、容量电价全面落地等政策的推进节奏存在不确定性。

六、结语

当前中国电力市场正处于改革深化、供需趋稳、电价筑底的关键周期。全国统一电力市场体系的加快建设、"十五五"5万亿电网投资的启动、算电协同新赛道的开辟，共同构成了电力行业投资的结构性机遇。

国电电力作为国家队龙头，凭借煤电一体化的成本优势、多能互补的电源结构和持续高分红的价值锚定，是电力板块中兼具防御性和成长性的核心标的。短期电价下行压力不改其长期配置价值。

宁波能源作为地方国企的精细化转型样本，在"收缩规模、提升质量"的战略指引下盈利显著改善，叠加宁波作为算电协同试点城市的区位红利，具备一定的差异化投资价值，但体量较小决定了其更适合作为组合中的弹性品种。

整体来看，电力板块当前市盈率约17.78倍、市净率约1.68倍，均处于历史中位偏低水平，安全边际较充足。电力板块逐步完成"重资产建设—稳定运营—现金流释放"的经营步调，经营现金流显著释放，分红规模持续提升。在利率下行的大背景下，电力公用事业的高股息、稳经营属性将愈发凸显配置价值。

Grok

中国电力市场的发展趋势总体积极但充满结构性转型挑战。 报告中描述的"十五五"改革深化、供需平衡偏紧、容量电价与电网巨额投资、算电协同等要素，符合电力行业从计划向市场、从传统火电主导向新能源+灵活性资源转型的长期逻辑。全国统一电力市场体系推进（现货全覆盖、分时电价市场化、复合输配电价）将提升资源配置效率，但也放大电价波动和竞争压力。AI/数据中心等新负荷成为重要增量，绿电+储能+算力融合开辟新增长点，而煤电的"兜底+容量补偿"角色在转型期仍具价值。

总体利好因素包括政策支持（容量电价上调、电网5万亿级投资）、需求韧性（用电量稳增5–6%）、技术驱动（算电协同）。挑战在于新能源消纳压力、电价筑底阶段的盈利波动、燃料成本不确定性，以及市场化下成本控制与灵活调节能力的比拼。板块估值处于历史中低位，高股息属性在利率下行环境中具备吸引力，但短期电价与煤价压力仍存。

国家队（中央企业，如国电电力）与地方国企（如宁波能源）的微妙关系及利益输送/转移

在中国电力行业，中央企业（央企）与地方国企的关系是层级协同+竞争并存的复杂生态，受国有资产管理体制（国资委、地方国资委）、电网调度统一性、区域发展不平衡等影响。并非简单的"利益输送"，而是政策导向下的资源再配置、产业链协同与绩效考核驱动的动态平衡。

  • 央企优势与"国家队"定位：如国家能源集团旗下的国电电力，拥有全产业链（煤-电-运）、跨区域布局、大规模装机和资金技术实力。它们常承担全国性战略任务（如特高压外送、煤电保供、碳达峰），更容易获得政策红利（如容量电价倾斜、大型新能源项目审批支持、跨省交易优先）。利益"转移"有时表现为集团内部资产注入/整合、长协煤保供、融资便利等，支持央企发挥规模效应和稳定器作用。
  • 地方国企的区域深耕：宁波能源等依托本地资源（热电联产、土地、区位政策），在综合能源服务、分布式项目、地方算力/零碳场景中更灵活。地方国资委考核更注重本地贡献（如就业、税收、能源安全），它们可能通过与央企/电网的合作获得技术/资金支持，但规模和抗风险能力较弱。

利益输送/转移的微妙机制（中性观察，非绝对）：

  • 向上集中与向下支持：央企通过集团平台整合优质资产、优先布局全国性项目，部分地方项目可能向央企倾斜以优化全国布局；同时，央企常通过合资、股权合作、技术输出支持地方发展，实现"输血"。
