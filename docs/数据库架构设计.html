🌍 GeoEnv — 地缘政治分析

geo_env_events — 主表

字段类型约束说明
idBIGINT UNSIGNEDPK AUTO_INCREMENT自增主键
user_idBIGINT UNSIGNEDNOT NULL用户 ID
event_timeDATETIMENOT NULL事件发生/报道时间
titleVARCHAR(255)NOT NULL事件标题
sourceVARCHAR(100)NULL来源（新华社、路透社等）
regionVARCHAR(100)NULL地区（中东、欧洲、亚太等）
country_involvedVARCHAR(255)NULL涉及国家（逗号分隔）
event_typeENUMNOT NULL DEFAULT 'other'conflict 冲突 / diplomacy 外交 / health 卫生 / economy 经济 / security 安全 / other 其他
severity_levelTINYINT UNSIGNEDNOT NULL DEFAULT 1严重程度 1-5
descriptionTEXTNULL事件详细描述
impact_summaryTEXTNULL对市场/投资的影响总结
risk_scoreDECIMAL(5,2)NULL综合风险评分 0-100
tagsVARCHAR(255)NULL标签（逗号分隔）
raw_contentMEDIUMTEXTNULL原始完整报道内容
created_atDATETIMEDEFAULT CURRENT_TIMESTAMP创建时间
updated_atDATETIMEDEFAULT CURRENT_TIMESTAMP ON UPDATE更新时间

impact_levelTINYINT UNSIGNED影响程度 1-5

📊 Price — 价格数据

价格数据模块独立于三个分析模块，用于记录大宗商品现货价格和斐波那契计算结果。数据录入页使用独立内联 JS（不共享 DB.js）。

price_spot_data — 现货价格表

唯一约束：UNIQUE KEY uk_price_code (code) — 每个代码只允许一条记录，新增同代码自动更新价格。

字段类型说明
idBIGINT UNSIGNED PK自增主键
user_idBIGINT UNSIGNED用户 ID（默认 1）
codeVARCHAR(50) UNIQUE品种代码（CU、AL、RB 等，唯一）
varietyVARCHAR(100)品种名称（沪铜、沪铝等）
modelVARCHAR(100)型号（1#电解铜等）
specVARCHAR(100)规格
qualityVARCHAR(100)质量标准
priceDECIMAL(12,2)价格
created_atDATETIME创建时间
updated_atDATETIME更新时间

price_calc_records — 斐波那契计算结果表

字段类型说明
idBIGINT UNSIGNED PK自增主键
user_idBIGINT UNSIGNED用户 ID（默认 1）
codeVARCHAR(50)品种代码
varietyVARCHAR(100)品种名称
price1 ~ price5DECIMAL(12,2)最多 5 个输入价格
result_jsonTEXT完整计算结果（JSON 编码）
created_atDATETIME创建时间

注意：price_calc_records 没有 updated_at 字段，计算结果保存后不可修改。无外键约束，独立于分析模块表。

💳 PerAcc — 个人账户管理

个人账户管理模块用于管理人员信息、券商账户、持仓明细和出入金记录。使用四级嵌套 FK 链（人员 → 券商账户 → 持仓 / 出入金）。数据录入页使用独立内联 JS，不共享 DB.js。

per_accounts — 人员主表

唯一约束：UNIQUE KEY uk_id_number (id_number) — 每人只允许录入一次，重复身份证号被拒绝。

字段类型说明
idBIGINT UNSIGNED PK自增主键
user_idBIGINT UNSIGNED用户 ID（默认 1）
nameVARCHAR(100)姓名
id_numberVARCHAR(50) UNIQUE身份证号（唯一）
phoneVARCHAR(50)手机号
backup_phoneVARCHAR(50)备用电话
notesTEXT备注
created_atDATETIME创建时间

per_account_brokers — 券商账户子表

唯一约束：UNIQUE KEY uk_broker_account (broker_name, account_no) — 全局唯一，同一券商下不允许重复资金账号。

字段类型说明
idBIGINT UNSIGNED PK自增主键
person_idBIGINT UNSIGNED FK→ per_accounts.id（CASCADE 级联删除）
broker_nameVARCHAR(100)券商名称（如中信证券、华泰证券）
account_noVARCHAR(100)账号
account_typeENUM('stock','futures')账户类型：券商/期货（2026-06-30 新增）
total_assetsDECIMAL(15,2)总资产/权益（2026-06-30 新增）：券商取资金股票JSON，期货取日结单客户权益
market_valueDECIMAL(15,2)持仓市值（2026-06-30 新增）：券商取股票市值，期货取保证金占用合计
available_amountDECIMAL(15,2)可用资金

per_account_positions — 持仓明细子表（2026-06 新增期货字段）

字段类型说明
idBIGINT UNSIGNED PK自增主键

🌍 GeoEnv — DDL

CREATE TABLE geo_env_events (
    id                BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    user_id           BIGINT UNSIGNED NOT NULL DEFAULT 1,
    event_time        DATETIME NOT NULL,
    title             VARCHAR(255) NOT NULL,
    source            VARCHAR(100) DEFAULT NULL,
    region            VARCHAR(100) DEFAULT NULL,
    country_involved  VARCHAR(255) DEFAULT NULL,
    event_type        ENUM('conflict','diplomacy','health','economy','security','other') NOT NULL DEFAULT 'other',
    severity_level    TINYINT UNSIGNED NOT NULL DEFAULT 1,
    description       TEXT DEFAULT NULL,
    impact_summary    TEXT DEFAULT NULL,
    risk_score        DECIMAL(5,2) DEFAULT NULL,
    tags              VARCHAR(255) DEFAULT NULL,
    raw_content       MEDIUMTEXT DEFAULT NULL,
    created_at        DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at        DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE geo_impact_assessments (
    id                BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    event_id          BIGINT UNSIGNED NOT NULL,
    asset_class       ENUM('oil','stock','currency','commodity','bond','other') DEFAULT NULL,
    industry          VARCHAR(100) DEFAULT NULL,
    impact_direction  ENUM('positive','negative','neutral','mixed') DEFAULT NULL,
    impact_level      TINYINT UNSIGNED DEFAULT NULL,
    expected_duration ENUM('short','medium','long') DEFAULT NULL,
    description       TEXT DEFAULT NULL,
    FOREIGN KEY (event_id) REFERENCES geo_env_events(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE geo_event_sources (
    id                BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    event_id          BIGINT UNSIGNED NOT NULL,
    source_name       VARCHAR(100) DEFAULT NULL,
    source_url        VARCHAR(500) DEFAULT NULL,
    author            VARCHAR(100) DEFAULT NULL,
    publish_time      DATETIME DEFAULT NULL,
    excerpt           TEXT DEFAULT NULL,
    FOREIGN KEY (event_id) REFERENCES geo_env_events(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

financial_data JSON 结构

{
  "2025-Q4": {
    "period": "2025-Q4",
    "period_label": "2025年报",
    "financial_data": {
      "current_assets": {
        "total": 1532178150.27,      // 子项之和
        "reported_total": 1532000000, // 报表中的流动资产合计（可选）
        "items": [
          {"name": "货币资金", "amount": 320520104.35},
          {"name": "应收账款", "amount": 18186346.76}
        ]
      },
      "liabilities": { ... },
      "fixed_assets": { ... }
    }
  },
  "2026-Q1": { ... }
}

每个周期独立存储，前端按周期横向对比展示。total 为自动计算的子项之和，reported_total 为报表中提取的合计值（如有）。

安全防护（2026-06 新增）

层级措施说明
① 网络隔离数据库监听 127.0.0.1:3306仅本地 PHP 进程可连接，拒绝外部直连。

④ CORS 白名单仅允许已知前端域名GitHub Pages 域名 + localhost，其他来源被 CORS 拦截。
⑤ 参数化查询PDO prepared statements所有 API 使用 PDO::prepare + 参数绑定，防止 SQL 注入。
