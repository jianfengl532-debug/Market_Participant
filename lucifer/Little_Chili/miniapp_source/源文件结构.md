## 页面路由汇总

### TabBar 底部导航（5 个主页面）

| 页面 | 路径 | 说明 |
|------|------|------|
| 🏠 首页 | `pages/index/main` | 店铺首页，自定义导航 |
| 📋 菜单 | `pages/menu/main` | 点餐菜单，自定义导航 |
| 🎫 会员 | `pages/member/main` | 会员中心，自定义导航 |
| 📦 订单 | `pages/order/main` | 订单列表，自定义导航 |
| 👤 我的 | `pages/user/main` | 个人中心，自定义导航 |

### 主要子包页面

| 子包 | 页面 | 功能 |
|------|------|------|
| **packShop** | 店铺主页 / 列表 / 地图 / 搜索 / 横幅 | 门店位置与展示 |
| **packUser** | 优惠券 / 会员卡 / 充值 / 兑换码 / 推广 / 次卡 / 抖音团购 | 用户资产与营销 |
| **packOrder** | 预订单 / 支付 / 快付 / 成功 / 订单详情 | 下单支付流程 |
| **packDomain** | 地址 / 押金 / 集点 / 拼团 / 退款 / 套餐 / 代付 / 首单 | 业务领域功能 |
| **packOther** | 注册 / 发票 / 搜索商品 / 推荐商户 / 积分抽奖 / 幸运游戏 | 辅助功能 |

## 技术特征

- **UI 框架：** uView UI 组件库
- **渲染模式：** WebView 渲染
- **分包加载：** 8 个子包（Shop / User / Order / Domain / Other / Menu / Libs / userPack）
- **权限需求：** 地理位置（定位门店）、地址管理
- **页面框架：** 自定义导航栏 + TabBar 底部导航
- **营销功能：** 幸运转盘、积分抽奖、拼团、集点活动、推广返佣
